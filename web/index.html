Project Info

Twitter4R v0.3.0 provides Twitter REST API access using a more Rubyish design than other Java-like Ruby libraries. Additional features include SSL, proxy and user agent ("source") customization. For example, you can have your statuses link back to your product page. AFAIK this isn't easily customizable in other Ruby libraries.

The Twitter4R project is released under the MIT License as a Ruby Gem.

Downloads

Visit the Rubyforge site to download the latest Twitter4R release. All versions are available as a Ruby Gem:

sudo gem install twitter4r
