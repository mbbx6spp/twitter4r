Twitter4Rv0.2.4

The Ultimate Open Source Ruby Library for the Twitter REST API

Follow Twitter4R on Twitter => http://twitter.com/twitter4r

Twitter4R News

Project Info

Twitter4R v0.2.4 now provides 100% Twitter REST API coverage for Ruby. Additional features include SSL, proxy and user agent ("source") customization.

The Twitter4R project is released under the MIT License as a Ruby Gem.

Downloads

Visit the Rubyforge site to download the latest Twitter4R release. All versions are available as a Ruby Gem:

sudo gem install twitter4r
